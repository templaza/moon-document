# Installation

Moon framework is simple and easy to install. Please follow these easy steps to get started with this amazing framework:

- Download the framework file \(.zip\) -\> [DOWNLOAD NOW](https://github.com/templaza/moon-framework/releases)
- Extract the Zip package, then you'll see different folders included (refer to file structure below)
- Log into your Moodle Site administrator
- In the menu, choose: Plugins -\> Install Plugin
- Click to upload the theme_moon (zip), block_moon_blog (zip), block_moon_galleries (zip), and local_moon (zip)
- Click on Upload & Install\. And you now have the framework installed\!

## Moon package's file structure: 

Here below is the file structure you can see after extracting the Moon framework package:

- link to download moodledata and demo file (text document)
- moon_moonframe.sql
- theme_moon (zip)
- block (folder) 
  - _block_moon_blog (zip)_
  - _block_moon_galleries (zip)_
- document (folder)
  - index.html
- local (folder)
  - _local_moon (zip)_
